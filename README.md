# 🎰 Mini Casino Web Game

![Mini Casino Banner](https://media.giphy.com/media/3o7TKz2eMXx8M3J7mU/giphy.gif)

## 🎲 О проекте

**Mini Casino** — это моя первая небольшая интерактивная веб-игра, созданная с использованием чистых технологий:

- 🟧 HTML5
- 🎨 CSS3
- ⚡ JavaScript

Проект представляет собой мини-казино в браузере с простыми игровыми механиками, анимациями и интерактивными элементами.

Цель проекта — изучение основ Frontend-разработки:
- работа с DOM
- обработка событий
- создание анимаций
- управление состоянием игры
- стилизация интерфейса

# 🕹️ Как запустить проект

### 1. Клонировать репозиторий

```bash
git clone https://github.com/username/mini-casino.git
```

### 2. Перейти в папку проекта

```bash
cd mini-casino
```

### 3. Запустить

Открыть файл:

```
index.html
```

в любом браузере.
