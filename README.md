# 🎰 Casino — Mini Web Casino

![Casino Logo](https://media.giphy.com/media/3o7TKz2eMXx8M3J7mU/giphy.gif)

## 📝 О проекте

**Casino** — это моя первая небольшая работа в Frontend-разработке.

Проект представляет собой мини-казино, созданное полностью с нуля при помощи:

- HTML5
- CSS3
- JavaScript

Главная цель проекта — практика создания интерактивного веб-приложения, изучение работы с DOM, событиями и игровой логикой.


---

🎰 **Игровая механика**

# 🚀 Запуск проекта

### Клонирование репозитория:

```bash
git clone https://github.com/anemens3-oss/Casino.git
```

Перейдите в папку проекта:

```bash
cd Casino
```

Запустите:

```
index.html
```

или откройте проект через Live Server в VS Code.
